# Presentation Widgets

Widgets are small, reusable UI building blocks that compose our views.

## Contents
* **Glassmorphic Cards**: Reusable cards with background blur and neon borders.
* **Scan Progress Bars**: Custom glowing progress rings or indicators.
* **Category Badges**: Visual tags representing file types (Images, Videos, Audio, Documents, Others).
* **ActionButton**: Cyberpunk styled tactile buttons with subtle touch feedback and glow.

## Guidelines
* Keep widgets highly modular, reusable, and small.
* Standardize design patterns by drawing all styling properties from `Theme.of(context)` or custom tokens in `AppTheme`.
